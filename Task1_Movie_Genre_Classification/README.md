# Movie Genre Classification

## Project Overview
This project predicts the genre of a movie based on its plot summary using Machine Learning and Natural Language Processing (NLP).

## Dataset
IMDb Genre Classification Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression

## Project Workflow
1. Imported the required libraries.
2. Loaded the IMDb movie dataset.
3. Preprocessed the text data.
4. Converted movie plots into numerical features using TF-IDF.
5. Trained a Logistic Regression model.
6. Predicted movie genres.
7. Evaluated the model using accuracy and classification report.

## Files
- movie_genre_classification.ipynb – Notebook containing the code.
- train_data.txt – Training dataset.
- test_data_solution.txt – Test dataset.
- description.txt – Sample movie descriptions.
- movie_genre_predictions.csv – Predicted genres.
